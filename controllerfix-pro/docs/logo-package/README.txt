ControllerFix Pro — Logo Package
=================================

FOLDERS

icon-mark/          The controller pictogram WITHOUT any text.
                     Use this for every small size (favicons, app icons,
                     avatars, buttons) since it never becomes illegible
                     or distorted when shrunk.
    transparent/     Full color, transparent background — most flexible
    white/           Solid white — for dark backgrounds / single-color use
    dark/            Solid dark navy (#10012A) — for light backgrounds/print
    dark-tile/       Full color icon on a solid dark-navy square tile
                     (used for favicon.ico and app icons)

wordmark-lockups/    Icon mark + the "CONTROLLERFIX PRO" wordmark set in
                     Orbitron, matching the brand style consistently:
                     CONTROLLER in white, FIX in the brand gradient
                     (violet #8A2BE2 → pink #FF69B4), PRO in white.
                     Use for site headers, letterheads, email signatures.
      - controllerfix-lockup-dark-bg.png     (dark background)
      - controllerfix-lockup-light-bg.png    (light background)
      - controllerfix-lockup-transparent.png (transparent, white text)

full-lockup-large-only/  Icon stacked above the "CONTROLLERFIX PRO"
                     wordmark (gradient FIX), as a single square mark —
                     the closest match to the original logo layout.
                     Provided at 512 / 1024 / 2048px only. Below 512px
                     the wordmark becomes too small to read clearly, so
                     use icon-mark/ instead for anything smaller.

favicons/            Ready-to-use favicon.ico (16/32/48px) and
                     apple-touch-icon.png (180px), built from the
                     clean icon mark so the artwork stays crisp.

svg/                 Vector master files (fully editable).
      - controllerfix-full-mark.svg    Icon + "CONTROLLERFIX PRO" wordmark.
                                        Text is rendered as true vector
                                        outlines (not a font reference),
                                        so it displays identically in any
                                        browser, editor, or print software
                                        with no font installation needed.
                                        FIX uses a real SVG gradient fill.
      - controllerfix-icon-only.svg    Same icon, no wordmark — for
                                        building your own lockups.
      - controllerfix-original-uploaded.svg   Your original upload, for reference

COLORS USED
  Background:      #10012A
  Gradient start:  #8A2BE2  (BlueViolet)
  Gradient end:    #FF69B4  (Hot Pink)
  White:           #FFFFFF

RULE OF THUMB
  Below 512px  →  use icon-mark/ (clean, no text)
  512px+ alone →  full-lockup-large-only/ is fine
  Need a wordmark next to the icon → wordmark-lockups/
  Need to edit the logo in Illustrator/Figma → svg/controllerfix-full-mark.svg
